# Product Sales Dashboard (Excel)

## Overview
Interactive Excel dashboard for analyzing product sales performance across categories and years.

## Key KPIs
- Total Sales
- Average Sales
- Total Orders
- Products Sold
- Category-wise Sales
- Product-wise Quantity
- Order Status Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables & Charts
- Slicers
- Excel Formulas
